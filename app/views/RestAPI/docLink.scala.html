Documentation for "links"

The "Link" profile object is a @Messages("shortName") object that defines a relationship between 2 resources.

  • Href is an absolute URL, that a client may follow to load this resource's content.
  • Rel is an absolute URL to a Profile definition for the target resource designed by href.
  • Title is the link title.

Sample:

{
    "link":
        {
            "href": "@ConferenceDescriptor.current().conferenceUrls.cfpURL/api/conferences/@ConferenceDescriptor.current().eventCode",
            "rel": "@ConferenceDescriptor.current().conferenceUrls.cfpURL/api/profile/conference",
            "title": "See more details about @Messages("longYearlyName")"
        }
}
}
