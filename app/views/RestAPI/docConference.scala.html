Devoxx API

Documentation for "conference"

Returns a JSON Array of Link object.

A conference is described by an eventCode, a label and a list of links to extra resources such as the list of speakers and the list of schedules.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showConference(Conference.devoxxFrance2014.eventCode).absoluteURL()

Result:

{
    "eventCode": "devoxxFR2014",
    "label": "Devoxx France 2014, 16 au 18 avril 2014",
    "links": [
        {
            "href": "http://cfp.devoxx.fr/api/conferences/devoxxFR2014/speakers",
            "rel": "http://cfp.devoxx.fr/api/profile/list-of-speakers",
            "title": "See all speakers"
        },
        {
            "href": "http://cfp.devoxx.fr/api/conferences/devoxxFR2014/schedules/",
            "rel": "http://cfp.devoxx.fr/api/profile/schedules",
            "title": "See the whole agenda"
        }
    ]
}
}
